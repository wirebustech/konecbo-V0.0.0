# **App Name**: Konecbo

## Core Features:

- Waitlist Registration: Allow researchers to sign up for the waitlist with their details (name, email, research interests).
- Product Write-up: Showcase the product details, explaining Konecbo's mission to connect researchers and accelerate discovery.
- Countdown Timer: Implement a countdown timer to a launch or key date.
- Dynamic Registration Count: Display a dynamically updating count of the number of people registered on the waitlist, creating social proof and excitement.

## Style Guidelines:

- Primary color: Dark Green.
- Background color: Very light cyan (#E5F3FF) to provide a clean and calming backdrop.
- Accent color: Orange.
- Headline font: 'Space Grotesk' sans-serif, for headlines. Body text: 'Inter', sans-serif, for body text
- Use clean, modern icons representing research, connection, and discovery.
- Clean and intuitive layout with a focus on readability and ease of navigation.
- Subtle animations for transitions and dynamic updates (e.g., the registration count increasing).
